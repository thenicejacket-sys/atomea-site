# Atomea — Site vitrine

Site one-page bilingue FR/EN d'**Atomea** (Automated Target Operating Model & Enterprise AI) — cabinet de conseil en transformation d'organisation, automatisation et IA d'entreprise.

## Caractéristiques

- **Monofichier** : `index.html` autonome (CSS, JS, photo en base64 inclus) — aucun build, aucun serveur requis
- **Bilingue** : bascule FR/EN, détection automatique de la langue du navigateur, choix mémorisé (localStorage)
- **Design « blueprint »** : panneaux marine quadrillés, annotations mono, animation de particules dans le hero (désactivée si `prefers-reduced-motion`)
- **Sections** : hero, frictions, 3 piliers, spécialité experts-comptables, méthode, réalisations, à propos, FAQ, contact
